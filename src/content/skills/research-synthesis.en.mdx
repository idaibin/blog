This skill should trigger when you need to turn many sources into one decision note.

- Extract claims from each source.
- Cluster repeated ideas.
- Mark uncertainty explicitly.
- Return a compact summary with next steps.

This is more than a prompt. It is a repeatable behavior block for research work.
